How do I access this project?

1. Head to "product-tracking".
2. Open Command Prompt in the folder.
3. Run "npx hardhat node"
4. You will get 20 ETH mock accounts with "Account #" and "Private Key"
5. Leave the terminal as it is for logs.
6. Open another CMD in the folder "product-tracking-frontend".
7. Change the "contractAddress" value.
7. Run the command "npm start".
8. Leave the terminal as it is for logs.
9. Open another CMD in the folder.
10. Run "npx hardhat run scripts/deploy.js --network localhost".
11. Head to "localhost:3000".
12. Confirm your Metamask wallet account.
